# Monitoring

## What's Included

| Namespace | Content |
| :---- | :---- |
| `prometheus` | Prometheus metrics poller & TSDB |
| `node-exporter` | Provides deep insights into cluster worker host VM health |
| `kube-state-metrics` | Simple details about api-resources and their readiness status |
| `alertmanager` | Notification system for cluster issues |
| `grafana` | Graphing & Dashboarding system |
