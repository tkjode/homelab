# Home Theatre Project

## What's Included

| Namespace | Content |
| :--- | :---- |
| `plex` | An attempt to run Plex in a cluster |
| `jellyfin` | Jellyfin for streaming |
| `sabnzbd` | Usenet NNTP downloader |
| `sonarr` | NZB Index Hunter for TV Shows |
| `sonarr-anime` | NZB Index Hunter for Anime |
| `radarr` | NZB Index Hunter for Movies |

## Stateful Data

- Configuration databases & API Keys (restart durable)
- Downloaded Media Content --> NAS via PersistentVolumes
